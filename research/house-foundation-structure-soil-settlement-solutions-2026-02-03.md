# โครงสร้างบ้านที่รองรับการทรุดตัว (House Foundation Structure for Soil Settlement)

**วันที่:** 3 กุมภาพันธ์ 2026
**ระดับ:** Standard Research
**สถานะ:** สมบูรณ์

---

## Executive Summary (สรุปภาพรวม)

เมื่อเผชิญกับปัญหาดินยุบหลังการถมดิน การแก้ปัญหาโดยใช้ **โครงสร้างบ้านที่ออกแบบรองรับการทรุดตัว** เป็นแนวทางที่ได้ผลจริงและเป็นที่ยอมรับในวงการวิศวกรรมโยธา

**Claim 1:** เสาเข็มต้องลงไปถึงชั้นดินเดิมที่มีความแข็งแรงเพียงพอ ไม่ฝากน้ำหนักบนดินที่เพิ่งถมใหม่
- **Confidence:** HIGH
- **Reason:** ได้รับการยืนยันจากแหล่งข้อมูลวิศวกรรมโยธาหลายแหล่ง
- **Sources:** [1][2][3][8][9]

**Claim 2:** ระบบพื้น Slab on Beam ช่วยให้ตัวบ้าน "ลอย" เหนือดินที่อาจทรุดตัว ลดความเสี่ยงของรอยร้าว
- **Confidence:** HIGH
- **Reason:** เป็นเทคนิคที่ได้รับการยอมรับและใช้งานจริงในโครงการที่มีปัญหาดินอ่อน
- **Sources:** [7][9][10]

**Claim 3:** การแยกโครงสร้างรอบนอกจากตัวบ้านหลักช่วยป้องกันการดึงรั้งเมื่อเกิดการทรุดตัว
- **Confidence:** HIGH
- **Reason:** เป็นหลักการออกแบบมาตรฐานในวิศวกรรมโยธา
- **Sources:** [6][7]

## เสาเข็มต้องถึงชั้นดินเดิม (Deep Foundation)

### หลักการสำคัญ

**หัวใจสำคัญ:** ปลายเสาเข็มต้องหยั่งลงไปใน **ชั้นดินเดิม** ที่มีความแข็งพอ (ชั้นดินเดิม) ไม่ใช่ฝากน้ำหนักไว้บนดินที่เพิ่งถมใหม่

```
┌─────────────────────────────────────┐
│           พื้นดินถมใหม่            │
│       (ไม่ใช้ฐานราก)              │
├─────────────────────────────────────┤
│           ดินผิว/ดินทราย          │
├─────────────────────────────────────┤
│         ชั้นดินเหนียว             │
├─────────────────────────────────────┤
│      ชั้นดินเดิม/ชั้นหิน ←─── ปลายเสาเข็มต้องถึงตรงนี้
└─────────────────────────────────────┘
```

### ประเภทเสาเข็มที่ใช้กับบ้านพักอาศัย

| ประเภทเสาเข็ม | คำอธิบาย | ความลึกแนะนำ | ข้อดี | ข้อเสีย |
|-------------|----------|-------------|-------|---------|
| **เสาเข็มไม้** | ใช้ไม้แก่ขนาดใหญ่ | 6-15 เมตร | ราคาถูก ติดตั้งเร็ว | ไม่ทนทานต่อความชื้น |
| **เสาเข็มปั้นจั่น (Spun Pile)** | คอนกรีตเสริมแรงหมุนเวียน | 8-20 เมตร | แข็งแรง ทนทาน | ราคาสูง ต้องใช้เครื่องจักร |
| **เสาเข็มเจาะ (Bored Pile)** | เจาะรูแล้วเทคอนกรีต | 10-30 เมตร | เหมาะกับพื้นที่จำกัด | ใช้เวลานาน ราคาสูง |
| **Micropile** | เสาเข็มขนาดเล็ก | 5-20 เมตร | เล็ก แข็งแรง พื้นที่น้อย | รับน้ำหนักได้น้อยกว่า |

### มาตรฐานการตอกเสาเข็ม

| พารามิเตอร์ | ค่าแนะนำ |
|------------|-----------|
| **ค่า N (SPT) ของชั้นดินรองรับ** | > 10 หรือชั้นหิน |
| **ความลึกเสาเข็มขั้นต่ำ** | 6 เมตร หรือผ่านชั้นดินอ่อน |
| **ระยะห่างระหว่างเสาเข็ม** | 2-4 เมตร (ขึ้นกับโหลด) |
| **กำลังรับน้ำหนักต่อเสา** | 30-100 ตัน (ขึ้นกับชนิดและขนาด) |

### ปัญหาที่พบบ่อยกับเสาเข็ม

**Claim 4:** ในหลายกรณีที่สำรวจบ้านทรุด พบว่าเสาเข็มมุมบ้านมีการทรุดตัวสูงสุดแม้จะรับน้ำหนักน้อยที่สุด ซึ่งเกิดจากเสาเข็มเสียหายหรือหัก
- **Confidence:** HIGH
- **Reason:** สอดคล้องกับข้อมูลจากแหล่งวิจัยวิศวกรรมโยธา
- **Sources:** [5]

**ปัญหาที่พบ:**
1. เสาเข็มไม่ถึงชั้นดินแข็ง
2. เสาเข็มเสียหายขณะตอก
3. ปลายเสาเข็มไม่ได้รับการป้องกันการกัดเซาะ
4. เสาเข็มไม่แน่นองเนื่องจากดินรอบๆ ทรุดตัว

## ระบบพื้น Slab on Beam (พื้นลอยบนคาน)

### หลักการทำงาน

**Slab on Beam** คือระบบพื้นที่พื้นคอนกรีต "ลอย" อยู่บนคาน (Beam) ซึ่งเชื่อมต่อกับเสาเข็ม ทำให้ตัวบ้านไม่ติดกับดินที่อาจทรุดตัว

```
┌─────────────────────────────────────┐
│           พื้นบ้าน (Slab)           │
│         ความหนา 10-15 ซม.         │
├─────────────────────────────────────┤
│         คานเสริมแรง (Beam)          │
│       ขนาด 30x50 ซม. หรือใหญ่กว่า   │
├─────────────────────────────────────┤
│            เสาเข็ม (Pile)            │
│              │││││                  │
│         ▼▼▼▼▼ ลงไปชั้นดินเดิม       │
└─────────────────────────────────────┘
```

### ข้อดีของระบบ Slab on Beam

| ข้อดี | คำอธิบาย |
|-------|----------|
| **ลอยเหนือดินทรุด** | ตัวบ้านไม่ตามดินยุบ หากดินรอบๆ ทรุดตัวลง ตัวบ้านจะยังคงอยู่ในระดับเดิม |
| **รับน้ำหนักได้ดี** | เหมาะกับโครงสร้างหนัก เช่น โรงงาน คลังสินค้า |
| **ป้องกันรอยร้าว** | ลดการแตกร้าวจากดินทรุดเนื่องจากตัวบ้านไม่ติดกับดิน |
| **ระบายน้ำได้** | มีช่องว่างใต้ถุนระบายน้ำ ลดปัญหาความชื้น |
| **เข้าถึงระบบไฟฟ้าได้ง่าย** | สายไฟและท่อประปาวางไว้ใต้พื้น เข้าถึงซ่อมแซมได้ง่าย |

### โครงสร้างที่เหมาะสมกับ Slab on Beam

**Claim 5:** Slab on Beam เหมาะกับพื้นที่ที่มีปัญหาดินอ่อนหรือดินที่ถมใหม่ สามารถป้องกันการทรุดตัวได้เกือบ 100%
- **Confidence:** HIGH
- **Reason:** ยืนยันโดยผู้เชี่ยวชาญด้านวิศวกรรมโยธา
- **Sources:** [9]

| สภาพพื้นที่ | เหมาะกับ Slab on Beam | เหตุผล |
|-----------|------------------------|--------|
| ดินถมใหม่ | ✅ เหมาะอย่างยิ่ง | ดินอาจทรุดตัวในอนาคต |
| ดินอ่อน | ✅ เหมาะอย่างยิ่ง | ดินไม่รองรับน้ำหนัก |
| พื้นที่น้ำท่วม | ✅ เหมาะ | มีช่องว่างใต้ถุนระบายน้ำ |
| ดินแข็ง | ❌ ไม่จำเป็น | ใช้ Slab on Grade ได้ |
| พื้นที่ชายเลน | ✅ เหมาะ | เสาเข็มต้องลึกพอ |

### ขนาดคานและพื้นที่แนะนำ

| ส่วนประกอบ | ขนาดแนะนำ | หมายเหตุ |
|-----------|------------|---------|
| **คาน (Beam)** | 30x50 ซม. ถึง 40x60 ซม. | ขึ้นกับระยะห่างเสา |
| **พื้น (Slab)** | 10-15 ซม. | ใช้เหล็กเสริมแรง จำนวนมาก |
| **ระยะห่างเสา** | 3-5 เมตร | ขึ้นกับโหลดและขนาดคาน |
| **ความสูงใต้ถุน** | 50-80 ซม. | เพื่อระบายน้ำและระบายอากาศ |

## ครีบกันดิน (Grade Beam / Ground Beam)

### วัตถุประสงค์

**ครีบกันดิน** หรือ **Grade Beam** เป็นคานคอนกรีตเสริมแรงที่วางอยู่เหนือผิวดินและก่อลงไปใต้ดิน 30-50 ซม.

```
     ผนังบ้าน
        │
        ├─────────── ครีบกันดิน (30-50 ซม.)
        │         ป้องกันดินไหลเข้า
    ────┴────    และปิดบังโพรง
   ดินถม │
        │
```

### ประโยชน์ของครีบกันดิน

| ประโยชน์ | คำอธิบาย |
|---------|----------|
| **ปิดบังโพรง** | ปิดบังโพรงที่อาจเกิดขึ้นในอนาคตจากการทรุดตัว |
| **ป้องกันสัตว์** | ป้องกันสัตว์เลื้อยคลานเข้าไปอาศัยใต้ถุนบ้าน |
| **กันดินถล่ม** | ช่วยยึดเกาะดินรอบๆ ไม่ให้ถล่มเข้ามา |
| **เสริมความแข็งแรง** | เชื่อมต่อเสาเข็มให้เป็นโครงสร้างเดียวกัน |

### รายละเอียดการก่อสร้างครีบกันดิน

| รายการ | รายละเอียด |
|--------|-------------|
| **ความลึกจากผิวดิน** | 30-50 ซม. |
| **ความกว้าง** | 20-30 ซม. |
| **ความสูง** | 30-50 ซม. |
| **เหล็กเสริมแรง** | 2-4 เส้น ขนาด DB12 หรือ DB16 |
| **คอนกรีต** | ผสมเหล็ก 3000 psi ขึ้นไป |

## แยกโครงสร้างรอบนอก (Separation Joint)

### หลักการ

**Separation Joint** หรือ **Expansion Joint** คือการแยกโครงสร้างส่วนขยาย เช่น ที่จอดรถ ลานซักล้าง ออกจากตัวบ้านหลัก เพื่อป้องกันการดึงรั้งเมื่อเกิดการทรุดตัว

```
┌──────────────┐   ←── Expansion Joint ──→   ┌──────────────┐
│  บ้านหลัก    │                                 │  ที่จอดรถ    │
│ (Slab on Beam)│                                 │ (Slab on Grade)│
│   เสาเข็ม    │                                 │   บนดินถม    │
└──────────────┘                                 └──────────────┘
```

### ประโยชน์ของการแยกโครงสร้าง

| ประโยชน์ | คำอธิบาย |
|---------|----------|
| **ป้องกันการดึงรั้ง** | หากส่วนรอบนอกทรุดตัว จะไม่ส่งผลต่อตัวบ้านหลัก |
| **ลดความเสี่ยงรอยร้าว** | รอยร้าวจากการทรุดตัวจะไม่ลามไปยังตัวบ้านหลัก |
| **บำรุงรักษาง่าย** | สามารถซ่อมแซมส่วนที่เสียหายได้โดยไม่กระทบตัวบ้าน |
| **ประหยัดค่าใช้จ่าย** | ส่วนรอบนอกสามารถใช้โครงสร้างที่ถูกกว่าได้ |

### วิธีการทำ Separation Joint

| วิธี | คำอธิบาย |
|-----|----------|
| **Polyethylene Foam** | ใส้ฟองน้ำหนา 1-2 ซม. กั้นกลางร่องต่อ |
| **Bituminous Strip** | แถบยางมะตอยกันน้ำรั่ว |
| **Joint Filler** | วัสดุอุดรอยต่อที่ยืดหยุ่นได้ |

## เปรียบเทียบ Slab on Beam กับ Slab on Grade

### ตารางเปรียบเทียบ

| ประเภท | Slab on Beam | Slab on Grade |
|--------|-------------|---------------|
| **คำอธิบาย** | พื้นวางบนคานที่เชื่อมกับเสาเข็ม | พื้นเทโดยตรงบนดินที่บดอัด |
| **ความเสี่ยงต่อดินทรุด** | **ต่ำมาก** (ลอยเหนือดิน) | **สูง** (ตามดินทรุด) |
| **ค่าใช้จ่าย** | สูงกว่า (เสาเข็ม + คาน + พื้น) | ถูกกว่า (เพียงพื้น) |
| **เหมาะกับ** | ดินอ่อน ถมดินใหม่ พื้นที่น้ำท่วม | ดินแข็ง มั่นคง ไม่ทรุดตัว |
| **ความทนทาน** | ยาวนาน (20-30 ปีขึ้นไป) | ขึ้นกับคุณภาพดิน (10-20 ปี) |
| **ระยะเวลาก่อสร้าง** | นานกว่า (2-3 เดือน) | เร็วกว่า (1-2 เดือน) |
| **การซ่อมแซม** | ยาก (ต้องระวังเสาเข็ม) | ง่ายกว่า |
| **ระบายน้ำ** | ดี (มีช่องว่างใต้ถุน) | ต้องอาศัยความลาดเอียง |
| **เข้าถึงระบบไฟฟ้า** | ง่าย (อยู่ใต้พื้น) | ยาก (ฝังในพื้น) |

### เมื่อใดควรใช้ Slab on Beam

**ใช้ Slab on Beam เมื่อ:**
1. ดินเป็นดินอ่อนหรือดินที่ถมใหม่
2. พื้นที่มีประวัติการน้ำท่วม
3. พื้นที่มีปัญหาดินทรุดตัว
4. ต้องการความมั่นคงสูงสุด
5. ตัวบ้านมีน้ำหนักมาก (หลังคาโค้ง ตกแต่งหนัก)

**ใช้ Slab on Grade เมื่อ:**
1. ดินแข็งและมั่นคง
2. พื้นที่ไม่มีประวัติการน้ำท่วม
3. งบประมาณจำกัด
4. ต้องการก่อสร้างเร็ว

## แนวทางการออกแบบที่เหมาะสม

### ขั้นตอนการออกแบบ

```
1. สำรวจพื้นที่ → 2. ทดสอบดิน (Soil Test) → 3. ออกแบบเสาเข็ม →
4. ออกแบบคาน → 5. ออกแบบพื้น → 6. ออกแบบระบบระบายน้ำ →
7. แยกโครงสร้างรอบนอก → 8. ตรวจสอบความปลอดภัย
```

### เกณฑ์การออกแบบ

| ขั้นตอน | รายละเอียด |
|---------|------------|
| **1. สำรวจพื้นที่** | ตรวจสอบระดับน้ำใต้ดิน ประวัติน้ำท่วม ดินบริเวณรอบๆ |
| **2. ทดสอบดิน** | SPT Test, Bore Hole ลึก 10-20 เมตร |
| **3. ออกแบบเสาเข็ม** | ปลายเสาเข็มต้องถึงชั้นดินที่มีค่า N > 10 |
| **4. ออกแบบคาน** | คำนวณโหลดและโมเมนต์ ใช้ Grade Beam เชื่อมต่อเสา |
| **5. ออกแบบพื้น** | ใช้ Slab on Beam พื้นหนา 10-15 ซม. |
| **6. ระบบระบายน้ำ** | วางท่อระบายน้ำใต้ดิน ทำความลาดเอียง 1-2% |
| **7. แยกโครงสร้าง** | ใช้ Expansion Joint แยกส่วนขยาย |
| **8. ตรวจสอบ** | ตรวจสอบความปลอดภัยและความเหมาะสม |

### งบประมาณโดยประมาณ

| รายการ | ค่าใช้จ่ายโดยประมาณ (บาท/ตร.ม.) |
|--------|-------------------------------|
| **เสาเข็มปั้นจั่น** | 800-1,500 |
| **คานเสริมแรง (Grade Beam)** | 200-400 |
